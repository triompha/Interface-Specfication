在进入系统时，使用本接口获取当前登录用户的信息。

本接口 *不得* 添加CSRF Token检验。

## 请求URL

    GET /users/current

## 通用参数

无参数

## 响应

### 用户已登录

状态码：200

响应格式：

    {
        authorityList: [ // 当前用户权限列表
            {string}, // 每一项为一个权限
            ...
        ],
        pageSize: {number}, // 表格的每页显示条目数
        sessionToken: {string}, // CSRF Token
        // 用户其它字段
    }

其中对象中写明的字段 **必须提供** 且 **不得改变字段名称** 。

### 用户未登录

状态码：403

无响应内容
